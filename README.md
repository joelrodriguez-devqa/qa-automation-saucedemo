# QA Automation - SauceDemo

Proyecto de automatización de pruebas utilizando Python y Selenium.

## Autor

Joel Rodríguez
QA Automation Trainee

## Descripción

Este script automatiza el proceso de login en SauceDemo y valida que la página cargue correctamente mediante la verificación de productos.

## Funcionalidades

* Login automático
* Validación de login exitoso
* Conteo de productos
* Verificación de carga de la página

## Tecnologías utilizadas

* Python
* Selenium

## Ejecución

1. Instalar dependencias:
   pip install -r requirements.txt

2. Ejecutar script:
   python test_login.py

## Resultado esperado

El script mostrará en consola:

* "Login exitoso"
* Número de productos encontrados
